# FILES-java
Exercicios de Java focados em manipulacao de arquivos e pastas.

## Sobre
Este repositorio reune praticas para leitura, escrita, criacao e organizacao
de arquivos e diretorios usando a API de I/O do Java.

## Objetivos
- Praticar `java.io` e `java.nio.file`
- Entender tratamento de excecoes em operacoes de arquivo
- Automatizar tarefas simples de organizacao de pastas

## Estrutura
- `src/` codigo fonte dos exercicios

## Requisitos
- JDK 8+ instalado
- IDE (opcional): IntelliJ, VS Code ou outra

## Como executar
1) Abra o projeto na sua IDE
2) Encontre a classe com o `application.Program`
3) Execute o arquivo desejado

Se preferir via terminal (ajuste o caminho da classe):
```bash
javac src/SeuExercicio.java
java -cp src SeuExercicio
```

## Exercicios (exemplos)
- Criar e ler arquivos de texto
- Listar e tratar dados de um arquivo
- Gerar relatorios simples a partir de arquivos .csv
